\begin{abstract}

Despite being able to produce reliable software, formal methods have hardly been adopted in mainstream programming. 
With the advent of large language models, it becomes more feasible to automatically generate code along with verification guarantees. 
This research explores whether LLMs can produce verified code from a textual description and a partial specification. 
We were able to achieve 63\% success rate in Nagini and \todo{this} in Viper on the HumanEval benchmark.
